## **RULES & CONTROLS:**
	-> Press ENTER to start
	-> Press P to Pause/Resume
	-> Press Q to quit playing and return to MENU
	-> Press R to restart a game (only in GAMEOVER state)
		(Cannot be restarted without quitting game)

	-> Key Mappings:
		-> LEFT ARROW - Move left
		-> RIGHT ARROW - Move right
		-> DOWN ARROW - Soft drop
		-> UP - Rotate clockwise
		-> Z - Rotate counter-clockwise
		-> SPACEBAR - Hard drop 

## **SCORING:**
	-> 1 point for each soft drop
	-> N points for hard drop of N cells
	-> Line Clear: 100 * L for 1 line
					300 * L for 2 lines
					500 * L for 3 lines
					800 * L for 4 lines
					where L = Level
### LEVEL is increased by 1 when lines cleared becomes a multiple of 10.

### With each increase in level, falling speed of tetrominos is decreased, by a factor of 0.75 of the previous value.

### The default FPS of the game is set at 60 FPS and initial drop interval of tetrominos is set to 500 ms.
